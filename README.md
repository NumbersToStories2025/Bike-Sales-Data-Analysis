# Bike-Sales-Data-Analysis
Exploratory analysis of bike sales data using Excel

## Project Overview
This project analyzes bike sales.
It explores trends based on age, income, and commute distance. 
[See reference to dashboard](Bike Sales Analysis Screenshot.PNG)

## Tools Used
-Microsoft Excel - Data Cleaning, Pivot Tables, Data Visualization

### Data Cleaning
In the data preparation phase, I performed the following tasks:
1. Data loading and inspection
2. Removing duplicates
3. Data cleaning and formatting

### Exploratory Data Analysis
The EDA phase asked the following questions: 
1. What are the sales trends by region?
2. What are the sales trends by different demographics?
3. What are the sales trends by commute distance?

### Data Analysis
Pivot Tables used to answer questions during EDA

### Results/Findings
1. United States leads bike sales 
2. Males buy more bikes than women
3. Most bikes are purchased for a short commute (0-1 miles)

### Recommendations
1. Focus on expanding products to women to increase sales for that demographic
2. Invest in marketing for the United States during peak season
